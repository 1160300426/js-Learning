# Day 2 
## HTML属性
### 常用标签属性
* <h1\>		:align
* <body\>	:bgcolor背景颜色
* <a\>		:target规定在何处打开连接
### 通用属性
* class :规定元素的类名
* id	:规定元素唯一ID
* style	:规定元素的样式
* title	:规定元素的额外信息
## HTML 的格式化
<table class="table table-bordered table-striped table-condensed">
	<tr>
	<td>标签</td>
	<td>描述</td>
	</tr>
	<tr>
	<td>< i></td>
	<td>定义斜体</td>
	</tr>
	<tr>
	<td> < b> </td>
	<td>定义粗体</td>
	</tr>
	<tr>
	<td>< big></td>
	<td>定义大号字体</td>
	</tr>
</table>
等等...
## HTML链接
### 连接数据：
* 文本链接
* 图片链接
### 属性：
* href属性：指向另一个文档的链接
* name属性：创建文档内的链接
### img标签属性：
* alt：替换文本属性
* width	：宽
* height：高
