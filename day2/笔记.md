# Day 2 
## HTML属性
### 常用标签属性
* <h1\>		:align
* <body\>	:bgcolor背景颜色
* <a\>		:target规定在何处打开连接
### 通用属性
* class :规定元素的类名
* id	:规定元素唯一ID
* style	:规定元素的样式
* title	:规定元素的额外信息
